Now we know that this is a swap file of Vim.

A bit of `man vim` teach us `-r {file}   Recovery mode.  The swap file is used to recover a crashed editing session.  The swap file is a file with the same filename as the text file with ".swp" appended.  See ":help recovery".`.

So :
```bash
user@kali:~/Documents/BesidesbosCTF/Misc/Swipe$ mv swipe swipe.swp
user@kali:~/Documents/BesidesbosCTF/Misc/Swipe$ mkdir /tmp/swipe
user@kali:~/Documents/BesidesbosCTF/Misc/Swipe$ vim -r swipe.swp 
Using swap file "swipe.swp"
"/tmp/swipe/flag.png" [New DIRECTORY]
Recovery completed. You should check if everything is OK.
(You might want to write out this file under another name
and run diff with the original file to check for changes)
You may want to delete the .swp file now.
```
We see that this is a PNG file, so we remove the "lorem ipsum" sentences and save it.

And finally open the picture to see a QR Code. Just go to decode it and we get the flag :`flag{swipe_right_on_vim_swap_soisoisoisoisoi}`.
